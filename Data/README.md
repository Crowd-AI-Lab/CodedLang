This folder contains the datasets used in **CodedLang**, including annotated reviews, a coded language dictionary, and seed examples.

### 1. CodeLang Dataset

- **File:** `CodedLang.csv`  
- **Description:**  
  Main dataset of Chinese online reviews with coded language annotations.

### 2. Coded Language Dictionary

- **File:** `Code_Dictionary.csv`
- **Description:**  
  A dictionary of coded expressions derived from annotations.

### 3. Coded Language Seed Samples

- **File:** `Coded_Language_Seed_Samples.csv`  
- **Description:**  
  A small set of **23 seed examples** collected from social media and news media.
